= archffmpeg Docker container

== Usage

[source, sh]
----
sudo docker run -v `pwd`:/work --rm kaosf/archffmpeg /usr/sbin/ffmpeg \
  -i /work/src.mp4 SOME_OPTIONS /work/dst.mp4
sudo chown `whoami`: dst.mp4
----

== Build image

[source, sh]
----
docker build . -t kaosf/archffmpeg
----
